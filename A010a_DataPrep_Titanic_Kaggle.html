Data Prep (Metadados - Variáveis Explicativas [Núméricas, Categóricas])¶

Função para gerar Dataframe de Metadados:

def pod_academy_generate_metadata(dataframe):
    """
    Gera um dataframe contendo metadados das colunas do dataframe fornecido.

    :param dataframe: DataFrame para o qual os metadados serão gerados.
    :return: DataFrame contendo metadados.
    """

    # Coleta de metadados básicos
    metadata = pd.DataFrame({
        'nome_variavel': dataframe.columns,
        'tipo': dataframe.dtypes,
        'qt_nulos': dataframe.isnull().sum(),
        'percent_nulos': round((dataframe.isnull().sum() / len(dataframe))* 100,2),
        'cardinalidade': dataframe.nunique(),
    })
    metadata=metadata.sort_values(by='percent_nulos',ascending=False)
    metadata = metadata.reset_index(drop=True)

    return metadata

Tratamento dos valores nulos - (Variáveis Numéricas)¶

O tratamento de valores nulos é uma etapa fundamental no pré-processamento de dados para análise de modelagem.

  • Dados nulos, ou "missing values", são comuns em muitos conjuntos de dados e podem surgir devido a erros, omissões ou a natureza intrínseca da coleda de dados.
  • A presença de valores nulos pode distorcer análises estatísticas, comprometer a integridade dos dados e resultar em modelos de aprendizado de máquina imprecisos ou inviesados.
  • Existem várias abordagens para lidar com esses valores, incluindo a exclusão de registros com dados faltantes, a imputação com estatísticas como a média ou mediana,
    ou o uso de algoritmos mais avançados para estimar os valores ausentes.
  • A escolha do método de tratamento depende da natureza dos dados, da quantidade de valores nulos e do contexto da análise, e é crucial para garantir a qualidade
    e confiabilidade dos insights derivados dos dados.

Substituindo nulos pela média da variável numérica:

# substituindo nulos pela média da variável numérica
abt_02 = abt_02.fillna(abt_02.select_dtypes(include=['float64', 'int64','int32']).mean())

Tratamento dos valores nulos - (Variáveis Categóricas)¶

  • Ao tratar valores nulos em variáveis categóricas, as abordagens comuns incluem a substituição por uma nova categoria, como
    • "Desconhecido",
    • imputação com o valor mais frequente (moda),
    • ou o uso de modelos preditivos para estimar os valores ausentes.
  • Em certos casos, a exclusão de registros com nulos pode ser considerada, especialmente se sua ocorrência for insignificante.
  • A escolha do método deve considerar o impacto na distribuição da característica e no contexto geral, e a expertise do domínio pode oferecer insigths valiosos para o tratamento adequado.

Substituindo nulos por "Desconhecido" para categóricas:

Tratamento das Variáveis Numéricas - Normalização¶

A normalização é uma técnica usada para reescalar valores de uma variável para que eles fiquem dentro de um intervalo específico, geralmente entre 0 e 1.

A equação para normalização é:

$ X_{\text{norm}} = \frac{X - X_{\text{min}}}{X_{\text{max}} - X_{\text{min}}} $¶

Onde:

  • $ X $ é o valor atual que você deseja normalizar.
  • $ X_{\text{min}} $ é o valor mínimo da característica no conjunto de dados.
  • $ X_{\text{max}} $ é o valor máximo da característica no conjunto de dados.
  • $ X_{\text{norm}} $ é o valor normalizado de $ X $.

Quando você aplica essa fórmula, garante que $ X_{\text{norm}} $ estará no intervalo [0,1].

Se você quiser normalizar em um intervalo diferente, como [a,b], pode ajustar a fórmula conforme necessário.

MinMaxScaler()

  • Transforme recursos dimensionando cada recurso para um determinado intervalo.
  • Este estimador dimensiona e traduz cada recurso individualmente, de modo que fique no intervalo determinado no conjunto de treinamento, por exemplo, entre zero e um.
  • Essa transformação é frequentemente usada como uma alternativa à escala de média zero e variância unitária.
  • MinMaxScaler não reduz o efeito dos valores discrepantes, mas os reduz linearmente em um intervalo fixo, onde o maior ponto de dados que ocorre
    corresponde ao valor máximo e o menor corresponde ao valor mínimo.

Data Prep (Enconding - Variáveis Categóricas)¶

Variáveis Categóricas de Baixa Cardinalidade - Criação de Variáveis Dummy (OneHot Enconding):¶

  • Uma técnica de pré-processamento usada para converter variáveis categóricas em um formato que pode ser fornecido aos algoritmos de aprendizado de máquina.
  • Para cada valor único (domínio) em uma variável categórica, o OneHot Encoding cria uma nova coluna binária, on "1" indica a presença desse valor e "0" indica a ausência.
  • Esta técnica é especialmente útil para algoritmos que não podem lidar diretamente com informações categóricas.
  • Variáveis de baixa cardinalidade
  • cardinalidade <= 10 (este valor de corte pode ser ajustado)

https://scikit-learn.org/stable/modules/generated/sklearn.preprocessing.OneHotEncoder.html

card_cutoff = 2
df_categ_onehot = metadata_df[(metadata_df['cardinalidade'] <= card_cutoff) & (metadata_df['tipo'] == 'object')]
