Conceito de Metadados

Funo para gerar Dataframe de Metadados:

In[10]:
def pod_academy_generate_metadata(dataframe):
    """
    Gera um dataframe contendo metadados das colunas do dataframe fornecido.

    :param dataframe: DataFrame para o qual os metadados sero gerados.
    :return: DataFrame contendo metadados.
    """

    # Coleta de metadados bsicos
    metadata = pd.DataFrame({
        'nome_variavel': dataframe.columns,
        'tipo': dataframe.dtypes,
        'qt_nulos': dataframe.isnull().sum(),
        'percent_nulos': round((dataframe.isnull().sum() / len(dataframe))* 100,2),
        'cardinalidade': dataframe.nunique(),
    })
    metadata=metadata.sort_values(by='percent_nulos',ascending=False)
    metadata = metadata.reset_index(drop=True)

    return metadata

Normalizaes / Padronizaes

MinMaxScaler()

  • Transforme recursos dimensionando cada recurso para um determinado intervalo.
  • Este estimador dimensiona e traduz cada recurso individualmente, de modo que fique no intervalo determinado no conjunto de treinamento, por exemplo, entre zero e um.
  • Essa transformao frequentemente usada como uma alternativa escala de mdia zero e varincia unitria.
  • MinMaxScaler no reduz o efeito dos valores discrepantes, mas os reduz linearmente em um intervalo fixo, onde o maior ponto de dados que ocorre corresponde ao valor mximo e o menor corresponde ao valor mnimo.

OneHot Encoding / Label Encoding

OneHotEncoder()

  • Codifique recursos inteiros categricos usando um esquema one-hot, tambm conhecido como one-of-K.
  • A entrada para este transformador deve ser uma matriz de nmeros inteiros, denotando os valores assumidos por caractersticas categricas (discretas).
  • A sada ser uma matriz esparsa onde cada coluna corresponde a um valor possvel de um recurso.
  • Supe-se que os recursos de entrada assumem valores no intervalo [0, n_valores).
  • Essa codificao necessria para alimentar dados categricos para muitos estimadores do scikit-learn, principalmente modelos lineares e SVMs com os kernels padro.
